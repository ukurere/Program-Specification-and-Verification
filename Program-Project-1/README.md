# SP-Lab-3
Lab 3. lex/yacc-based parser

На базі Lex/YACC (або аналога) розробити синтаксичний аналізатор (з генерацією коду або обчисленнями - опціонально) для С або іншої сучасної імперативної (ООП, процедурної, тощо) мови програмування, або для арифметичних виразів (спрощений варіант).

Оцінювання: 
2 бала - коректний розвʼязок (працюючий проєкт)
2 бала - відповіді на питання по коду
+2 бала - для граматики мови C або C++
+2 бала - за відповіді на питання по теорії: LALR-аналізатор, принципи роботи yacc, як розвʼязуються конфлікти (shift-reduce, reduce-reduce) в ході розбору...
+2 бала - (рекомендовано) за додавання семантики (генерації коду) до AST (реалізацію обчислень)
+2 бала - за генерацію коду на цільовій мові в результаті трансляції
+2 бала - (рекомендовано) за візуалізацію AST (d3.js, або будь-яким зручним для користувача та наочним засобом, бажано в веб-браузері, засобами графіки в JS)
+2 бала - за завчасне здавання роботи (за тиждень до deadline)
за невчасну здачу - 50% від отриманої оцінки
при виявленні плагіату (суттєвих запозичень в коді) - оцінка 0 балів за всю роботу
